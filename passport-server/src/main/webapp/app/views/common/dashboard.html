Role Control

Roles enables you to assign specific permissions to a group and to fine-tune the security, use and administration of modules. Users assigned to the role or group, are granted those permissions as assigned to the role. Common examples of roles used with which you may be familiar include: anonymous, user, administrator and developer.

Http Session

An HttpSession object can hold conversational state across multiple requests from the same client. In other words, it persists for an entire session with a specific client. We can use it to store everything we get back from the client in all the requests the client makes during a session.
